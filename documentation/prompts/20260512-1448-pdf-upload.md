# PDF Upload (replace JPEG)

## Goal
Replace JPEG score sheet upload with PDF upload. Convert first page to JPEG at 300dpi before sending to Claude for extraction. Store the original PDF on disk.

## Key decisions
- PDF-only (JPEG support dropped entirely)
- Conversion via `pdftoppm` (Poppler CLI, available on macOS and Docker/Linux)
- JPEG is ephemeral — produced in `/tmp`, read into buffer, deleted immediately
- PDF stored on disk as `{hash}.pdf`; hash computed from PDF buffer
- 5MB file size limit unchanged
- Existing JPEG files in DB left as-is (forward-only migration)
- Deployment target: Fly.io (Docker) — `poppler-utils` installed via `apt-get`

## Backend changes

### `score-sheet.controller.ts`
- MIME filter: `image/jpeg` → `application/pdf`
- Error message: "Only PDF files are allowed"

### `score-sheet.service.ts`
- Added `convertPdfToJpeg(pdfBuffer: Buffer, hash: string): Promise<Buffer>`:
  1. Write PDF to `/tmp/{hash}.pdf`
  2. Run `pdftoppm -r 300 -jpeg -singlefile /tmp/{hash}.pdf /tmp/{hash}-out`
  3. Read `/tmp/{hash}-out.jpg` back into buffer
  4. Delete both temp files in `finally` (even on error)
  5. If `pdftoppm` fails: throw `BadRequestException('Invalid or unreadable PDF')`
- Uses `promisify(execFileCb)` from `child_process` (not `exec` — no shell injection risk)
- `extract()` now calls `convertPdfToJpeg()` before `callClaude()`; PDF buffer still passed to `fileService.persist()`

### `file.service.ts`
- Storage path: `{hash}.jpg` → `{hash}.pdf`

### `score-sheet.service.spec.ts`
- `jest.mock('child_process')` — default mock calls callback with `(null, '', '')`
- `jest.mock('fs/promises')` — `readFile` returns `Buffer.from('jpeg-bytes')`
- Added test: `throws BadRequestException when pdftoppm fails` — overrides execFile mock to call `cb(new Error(...))`
- Added eslint-disable for `no-unsafe-member-access`, `no-unsafe-assignment`, `no-unsafe-call` (jest.requireMock returns any)

### `file.service.spec.ts`
- Updated expected path from `abc.jpg` → `abc.pdf`

## Frontend changes

### `Upload.tsx`
- `accept="image/jpeg"` → `accept=".pdf"`
- Label: "Score sheet (JPEG)" → "Score sheet (PDF)"
- Client-side validation: `f.type !== 'image/jpeg'` → `f.type !== 'application/pdf'`
- Toast error: "Only JPEG files are allowed" → "Only PDF files are allowed"
